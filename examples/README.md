# Examples
Here, we provide `tutorial.ipynb` to demonstrate the two primary features of the `xtalmet` package: 1) calculating distances between crystals, and 2) evaluating the VSUN (Validity, Stability, Uniqueness, and Novelty) of a given set of crystals.
The `workshop`/`journal` directories contain all the code needed to reproduce the results presented in our papers:
- `workshop`: "Continuous Uniqueness and Novelty Metrics for Generative Modeling of Inorganic Crystals," presented at the NeurIPS 2025 AI4Mat workshop.
- `journal`: "Continuous SUN (Stable, Unique, and Novel) Metric for Generative Modeling of Inorganic Crystals," submitted to IOP Machine Learning: Science and Technology.
