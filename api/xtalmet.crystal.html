This module defines the Crystal class to store information about a single crystal.

Container for a single crystal structure.

Initialize a Crystal object.

The parameters are the same as those used in the __init__() method of the pymatgen.core.Structure class.

Parameters:

  • lattice (Lattice/3x3 array) ‚Äì The lattice, either as a pymatgen.core.Lattice or simply as any 2D array. Each row should correspond to a lattice vector. e.g. [[10,0,0], [20,10,0], [0,0,30]] specifies a lattice with lattice vectors [10,0,0], [20,10,0] and [0,0,30].

  • species ([Species]) ‚Äì

    Sequence of species on each site. Can take in flexible input, including:

    1. A sequence of element / species specified either as string

      symbols, e.g. [‚ÄúLi‚Äù, ‚ÄúFe2+‚Äù, ‚ÄúP‚Äù, ‚Ä¶] or atomic numbers, e.g. (3, 56, ‚Ä¶) or actual Element or Species objects.

    2. List of dict of elements/species and occupancies, e.g.

      [{‚ÄúFe‚Äù : 0.5, ‚ÄúMn‚Äù:0.5}, ‚Ä¶]. This allows the setup of disordered structures.

  • coords (Nx3 array) ‚Äì list of fractional/Cartesian coordinates of each species.

  • charge (int) ‚Äì overall charge of the structure. Defaults to behavior in SiteCollection where total charge is the sum of the oxidation states.

  • validate_proximity (bool) ‚Äì Whether to check if there are sites that are less than 0.01 Ang apart. Defaults to False.

  • to_unit_cell (bool) ‚Äì Whether to map all sites into the unit cell, i.e. fractional coords between 0 and 1. Defaults to False.

  • coords_are_cartesian (bool) ‚Äì Set to True if you are providing coordinates in Cartesian coordinates. Defaults to False.

  • site_properties (dict) ‚Äì Properties associated with the sites as a dict of sequences, e.g. {‚Äúmagmom‚Äù:[5, 5, 5, 5]}. The sequences have to be the same length as the atomic species and fractional_coords. Defaults to None for no properties.

  • labels (list[str]) ‚Äì Labels associated with the sites as a list of strings, e.g. [‚ÄòLi1‚Äô, ‚ÄòLi2‚Äô]. Must have the same length as the species and fractional coords. Defaults to None for no labels.

  • properties (dict) ‚Äì Properties associated with the whole structure. Will be serialized when writing the structure to JSON or YAML but is lost when converting to other formats.

Returns:

None

descriptions for args are copied from pymatgen.core.Structure class.
